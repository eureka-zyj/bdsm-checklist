# BDSM Checklist（调查用）v1.0

一个纯前端、静态网页实现的 BDSM 取向与偏好问卷，用于 **成年人之间的自我了解与沟通**。  
本项目不会上传或保存任何数据，所有填写内容只在当前浏览器本地存在，适合分享给朋友自行填写、讨论使用。

> ⚠️ 声明：  
> 本项目仅面向 **成年用户**，用于个人自我记录或伴侣之间沟通喜好，不鼓励或涉及任何违法、不安全或非自愿行为。请务必在 **知情同意 / 安全 / 理性 / 自愿** 的前提下使用（SSC / RACK）。

## ✨ 主要特性（Features）

- **多页面向导式填写**
  - 首页展示标题与说明，点击「下一步」进入填写流程。
  - 第二页填写基本信息：  
    - 填表人  
    - 年龄  
    - 生理性别（下拉 + 自定义选项）  
    - 性取向（下拉 + 自定义选项）  
    - 感情状态（下拉 + 自定义选项）  
    - 性经历人数  
  - 填表日期自动记录为当天，不需要手动输入，但会在结果中显示。

- **多大类分页面的项目表单**
  - BDSM 相关的大类被拆分成多个页面（如：束缚、体液及其功能化、恋物癖、羞辱、打/刑罚 Play、多人/非专一、角色扮演、感官刺激、服侍与控制、性行为与入体、窥视/露出 等）。
  - 页面顶部有 **横向导航条**，可以随时点击切换大类，当前大类高亮显示。
  - 每一类中的每个项目均采用表格形式展示。

- **图形化按钮选择**
  - 每个项目包含四列：
    1. **项目名称**
    2. **是否尝试过**：「是 / 否」按钮
    3. **喜欢程度**：1～5 的评分按钮  
       - 说明：  
         - 1 = 非常不喜欢  
         - 2 = 没有乐趣  
         - 3 = 一般 / 还行  
         - 4 = 喜欢  
         - 5 = 非常享受  
    4. **角色**：「主动 / 被动 / 双向 / 都不想」
  - 点击按钮即可高亮选中，同一组内互斥。

- **一键填充功能**
  - 每个大类表头下方提供“一键填满”按钮，对该大类内所有项目生效：
    - 一键「全部=是」或「全部=否」
    - 一键喜欢程度 = 1 / 2 / 3 / 4 / 5
    - 一键角色 = 主动 / 被动 / 双向 / 都不想
  - 方便用来快速给出一个“基准状态”，再逐项微调。

- **不强制全部填写**
  - 即使某些项目未选择，也可以继续下一步，直到最后一页。
  - 未填写的项目在结果中会明确标记为「（未填写）」或「未填写」。

- **结果导出**
  - 最后一页会显示「感谢填写！」，并提供两种导出方式：
    1. **生成文字结果**  
       - 以文本形式汇总：标题、填表时间、基本信息，以及按大类分组的项目结果。  
       - 每条结果示例：  
         - `肢体束缚（肢体捆缚帶）：尝试过；非常享受；作为被动对象`
       - 未填写时显示：`（未填写）`
       - 文本框内容可以直接复制保存。
    2. **生成 Excel 结果（实际为 CSV with .xls 扩展名）**  
       - 浏览器会下载一个 `BDSMchecklist_result.xls` 文件。  
       - 文件内部为 UTF-8 带 BOM 的 CSV，Excel 可直接打开。  
       - 结构：
         - 顶部是基本信息表（填表人、日期、年龄、性别、性取向、感情状态、性经历人数）。
         - 下方是明细表：  
           `大类, 项目, 是否尝试过, 喜欢程度, 角色`  
       - 未填写的项会显示为「未填写」。
       - 若 Excel 提示「文件格式与扩展名不符」，选择「是」即可正常打开。

- **纯前端实现**
  - 无后端、无数据库，不会将数据发送到服务器。
  - 所有逻辑（页面切换、校验、导出等）都在浏览器内完成。
